# Colorful Capacious Chests

A Wayward mod that adds colorful, high-capacity chest variants for better organization and increased storage.

## Crafting

To craft a colored chest, simply add **1 pigment item** to the standard chest recipe:
- Standard chest recipe + 1 pigment = Colored chest with enhanced capacity

Pigments can be any item from the color groups listed above. For example:
- Wooden Chest + Charcoal = Black Wooden Chest
- Copper Chest + Carrot = Orange Copper Chest
- Iron Chest + Crowberries = Blue Iron Chest

## Compatibility

- **Wayward Version:** 2.15.1-beta or higher
- **Multiplayer:** Fully compatible

npm run build

# Clean build artifacts
npm run clean
